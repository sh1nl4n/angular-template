# Angular Template Generator

## Build && Install

### powershall

`npm run build; npm install -g .`

### linux bash

`npm run build && npm install -g .`

## Use

Operators:

-   Generate (generate | g)

Structure:

-   Component (component | c)
-   Entity (entity | e)

Command:

-   `-v` `--verbose` - enable log args
-   `-p` `--path` - select path to generate

Command Use

`mes g c Menu -p ./src/pages`
